# Contar Vocales

## Contexto
Contar caracteres específicos en una cadena es una tarea fundamental en el procesamiento de texto. Las vocales (a, e, i, o, u) son a menudo de interés en el análisis lingüístico.

## Tarea
Escribe una función `count_vowels(s: str) -> int` que devuelva el número de vocales (tanto mayúsculas como minúsculas) en una cadena dada.

## Especificaciones
- **Entrada**: Una cadena `s` de longitud arbitraria.
- **Salida**: Un entero que representa el conteo total de vocales ('a', 'e', 'i', 'o', 'u' sin distinguir mayúsculas de minúsculas).

## Restricciones
- No uses `input()` ni `print()`.
- La cadena puede contener caracteres no alfabéticos.

## Ejemplo
```python
>>> count_vowels("Hello World")
3
>>> count_vowels("AEIOU")
5
```
