# Suma de Matrices

## Contexto
La suma de matrices es la operación de sumar dos matrices sumando las entradas correspondientes. Ambas matrices deben tener el mismo número de filas y columnas.

## Tarea
Escribe una función `matrix_add(mat1, mat2)` que calcule la suma elemento a elemento de dos matrices.

## Especificaciones
- La función toma dos listas 2D `mat1` y `mat2`.
- Devuelve una nueva lista 2D que contiene las sumas.
- Puedes asumir que ambas matrices siempre tienen las mismas dimensiones válidas y no están vacías.

## Restricciones
- Las matrices de entrada tendrán tamaños idénticos.
- Los valores pueden ser enteros o flotantes.

## Ejemplo
```python
>>> matrix_add([[1, 2], [3, 4]], [[5, 6], [7, 8]])
[[6, 8], [10, 12]]
```
