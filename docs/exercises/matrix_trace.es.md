# Traza de una Matriz

## Contexto
La traza de una matriz cuadrada se define como la suma de los elementos de la diagonal principal (desde la parte superior izquierda hasta la parte inferior derecha).

## Tarea
Escribe una función `matrix_trace(mat)` que calcule la traza de una matriz cuadrada dada.

## Especificaciones
- La función toma una lista 2D `mat`.
- Devuelve un solo número que representa la traza.

## Restricciones
- La matriz de entrada será cuadrada ($N \times N$).
- $N \ge 1$.

## Ejemplo
```python
>>> matrix_trace([[1, 2], [3, 4]])
5
```
