# Ejercicio: Inversión de Cadena

*[Read in English](README.md) | [Solve Online](https://github.dev/matcom/gym/blob/main/exercises/string_reversal/solution.py)*

## Antecedentes/Motivación

Invertir una cadena es una operación común que se puede lograr de varias maneras, como usar bucles o las capacidades integradas de Python como el rebanado. Este ejercicio ayuda a comprender la manipulación de cadenas y la inmutabilidad.

## La Tarea

Implementa una función `string_reversal(s: str) -> str` que invierta una cadena dada.

### Especificaciones

-   **Nombre de la Función**: `string_reversal`
-   **Argumentos**: `s` (str)
-   **Tipo de Retorno**: `str`
-   **Resultado Esperado**: La cadena invertida.

### Restricciones

-   La cadena de entrada puede estar vacía o contener cualquier carácter.

### Ejemplo

```python
>>> string_reversal("hello")
"olleh"
>>> string_reversal("Python")
"nohtyP"
```

## Instrucciones

1.  Abre `exercises/string_reversal/solution.py`.
2.  Implementa la función `string_reversal`.
3.  Cambia `SUBMIT = False` a `SUBMIT = True` en la parte superior del archivo cuando estés listo para ser evaluado.
4.  Ejecuta `python solution.py` localmente para verificar tu solución con las pruebas integradas.
