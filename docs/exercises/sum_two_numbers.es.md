# Ejercicio: Suma de Dos Números

*[Read in English](README.md) | [Resolver en Línea](https://github.dev/matcom/gym/blob/main/exercises/sum_two_numbers/solution.py)*

## Antecedentes/Motivación

La suma es una de las operaciones más fundamentales en la aritmética y la informática. Es la base para operaciones más complejas como la multiplicación y algoritmos avanzados. En este ejercicio, aprenderás a trabajar con argumentos numéricos y valores de retorno.

## La Tarea

Escribe una función `sum_two_numbers(a, b)` que tome dos enteros como entrada y devuelva su suma.

### Especificaciones

- **Nombre de la Función**: `sum_two_numbers`
- **Argumentos**:
    - `a` (int): El primer número.
    - `b` (int): El segundo número.
- **Valor de Retorno**:
    - `int`: La suma de `a` y `b`.

### Restricciones

- Los números pueden ser positivos, negativos o cero.
- Los números estarán dentro del rango de un entero estándar.

### Ejemplo

```python
>>> sum_two_numbers(3, 4)
7
>>> sum_two_numbers(-1, 1)
0
```

4. Ejecuta `python solution.py` para verificar con las pruebas integradas.
